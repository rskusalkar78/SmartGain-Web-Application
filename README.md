# SmartGain-Web Application

A modern, intelligent weight gain planning application built with React, TypeScript, and Tailwind CSS. SmartGain provides science-backed calorie calculations, personalized macro recommendations, and adaptive workout planning for healthy weight gain.

## Features

- **Intelligent Calorie Calculator**: Personalized daily calorie recommendations based on your body metrics and goals
- **Macro Planning**: Optimal protein, carb, and fat distribution for healthy weight gain
- **Adaptive Workouts**: Customized exercise plans that evolve with your progress
- **Science-Based**: Built on established nutrition science and metabolic research
- **Beginner-Friendly**: Designed for users new to weight gain planning
- **No Supplements Required**: Focus on whole foods and natural approaches

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn/ui components
- **Animations**: Framer Motion
- **Testing**: Vitest, Testing Library
- **Routing**: React Router DOM
- **State Management**: TanStack Query

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Open http://localhost:5173 in your browser

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run test` - Run tests
- `npm run lint` - Run ESLint
